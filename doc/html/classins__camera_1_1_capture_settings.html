Detailed Description

a class wrapping capture settings. CaptureSettings holds the temperary settings value you want to apply to the camera. The exactly settings to be applied are implied by a list of SettingType,which you may get by calling GetSettingTypes(). The types will be automatically recorded when you call any SetXXX() function. However, you can also ResetSettingTypes() or UpdateSettingTypes() mannually.

available settings and value range:

  • Contrast: 0~256, default 64
  • Saturation:0~256, default 64
  • Brightness:-256~256, default 0
  • WhiteBalance: see #PhotographyOptions_WhiteBalance
  • Sharpness: 0~6, default 3
